# Data

The datasets used in this project are not stored in this repository because of their size.

To run the notebook, download the required CSV files and place them in the `data/` folder.

## Required datasets

1. **OpenPowerlifting dataset**

   Source: https://www.kaggle.com/datasets/open-powerlifting/powerlifting-database?select=openpowerlifting-2024-01-06-4c732975.csv

   Expected filename:

   ```text
   openpowerlifting-2024-01-06-4c732975.csv
   ```

2. **Countries and continents dataset**

   Source: https://www.kaggle.com/datasets/hserdaraltan/countries-by-continent

   Expected filename:

   ```text
   countries and continents.csv
   ```

## Expected local file structure

```text
data/
├── openpowerlifting-2024-01-06-4c732975.csv
└── countries and continents.csv
```

## Note

The CSV files are ignored by Git and should not be uploaded to this repository.
